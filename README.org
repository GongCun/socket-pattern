#+TITLE: TCP并发服务器示例
- 安装
  #+BEGIN_SRC sh
  make -k
#+END_SRC

- 说明
  服务端口为 9989，直接启动服务器进程即可，无需任何参数。说明如下：
  | 名称       | 作用                                                         |
  |------------+--------------------------------------------------------------|
  | tcpserver  | 使用 select 的迭代服务器                                     |
  | tcpserver2 | 多进程服务器，为每个客户现场启动一个进程                     |
  | tcpserver3 | 多线程服务器，为每个客户现场启动一个线程                     |
  | server     | 将已连接套接字传递给子进程，子进程为每个客户现场启动一个线程 |
